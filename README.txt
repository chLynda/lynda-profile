LYNDA CHETTI — DIGITAL PROFILE

This package contains a mobile-first digital profile page designed to be accessed through a QR code.

FILES
- index.html: the profile page
- README.txt: instructions

WHAT TO UPDATE
1. Replace the LinkedIn placeholder links:
   Search for: https://www.linkedin.com/
   Replace with your real LinkedIn URL.

2. Add your CV:
   Put your PDF CV in the same folder and name it:
   CV_Lynda_CHETTI.pdf
   Or replace the link in the HTML with your hosted CV URL.

3. Optional:
   Add a GitHub / portfolio link in the Quick Access section if you want.

HOW TO USE WITH A QR CODE
- Host index.html online using GitHub Pages, Netlify, Google Drive public page, or another hosting option.
- Copy the public URL.
- Generate a QR code from that URL.
- Place the QR code on your CV with the label:
  Digital Profile
  Scan to access my CV, projects and professional links.
